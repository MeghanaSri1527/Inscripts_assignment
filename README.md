# 💼 Inscripts Internship Assignment

This is a front-end web application developed as part of the internship assignment for **Inscripts**. It is built using **React (TypeScript)**, **Vite**, and styled with **Tailwind CSS**.

---

## 🚀 Tech Stack

- ⚛️ React
- 🟦 TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🧹 ESLint

---
